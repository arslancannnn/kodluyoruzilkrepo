## 3. SORU

#### 1. UPDATE
İd'si 5 olan çalışanın isim ve email bilgilerinin değiştirlmesi

````SQL

UPDATE employee 
SET name = 'eren',
    email= 'eren_bakıcı@gmail.com'
WHERE id =5
RETURNING *;

````

#### 2. UPDATE
'A' harfi ile başlayan çalışanların ismini'XXXX' emailini ise null olarak değiştirilmesi
````SQL

UPDATE employee 
SET name = 'XXXX',
    email= NULL
WHERE name LIKE 'A%'
RETURNING *;

````

#### 3. UPDATE
İD değeri 13 ile 15 arasında olan çalışanların birthday bilgisinin null olarak değiştirilmmesi
````SQL

UPDATE employee 
SET birthday= NULL
WHERE id BETWEEN 13 AND 15
RETURNING *;

````

#### 4. UPDATE
'D' harfi ile başlayan ve  1 haziran 1970 öncesi doğan çalışanların isim bilgisinin 'XXXX' olarak değiştirilmesi
````SQL

UPDATE employee 
SET name = 'XXXX'
WHERE name LIKE 'D%' AND birthday<'1970-06-01'
RETURNING *;

````

#### 5. UPDATE
İsmi 'ie' ile biten ve email adresi 'lulu.com' olan çalışanların isminin 'XXXX' olarak değiştirilmesi
````SQL

UPDATE employee 
SET name = 'XXXX'
WHERE name LIKE '%ie' AND email LIKE '%lulu.com'
RETURNING *;

````

### 1. DELETE


### 2. DELETE
1955 öncesi doğan çalışanların silinmesi
````SQL

DELETE FROM employee
WHERE birthday < '1955-01-01'
RETURNING *;

````

### 3. DELETE
Baş harfi 'C' olan ve 2000'den sonra doğmuş çalışanların verilerinin silinmesi
````SQL

DELETE FROM employee
WHERE name LIKE 'C%' AND birthday > '2000-01-01'
RETURNING *;

````
