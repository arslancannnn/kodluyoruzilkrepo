## 1. Soru

````SQL

SELECT rating, COUNT(*) FROM  film
GROUP BY rating;

````

## 2. Soru

````SQL

SELECT replacement_cost, COUNT(*) FROM  film
GROUP BY replacement_cost
HAVING COUNT(*)>50
ORDER BY COUNT(*) ASC;

````

## 3. Soru

````SQL

SELECT store_id , COUNT(*) FROM  customer

````

## 4. Soru

````SQL

SELECT country_id , COUNT(*) FROM  city
GROUP BY country_id
ORDER BY COUNT(*) DESC
LIMIT 1;

````
